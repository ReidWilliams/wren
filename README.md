# Wren

A brutally small macOS native app for chronological scratch notes. Starts instantly and has keyboard shortcuts to rapidly seek backwards to earlier notes.

## Installation

### Pre-built

Download or copy `Wren.app` into `/Applications`.

### Build from source

Requires Xcode command line tools (`xcode-select --install`). Then build with

```
make
```

## Note History

Browse notes chronologically with keyboard shortcuts:

| Shortcut | Action |
|---|---|
| `⌘N` | New note |
| `⌘[` | Go back one note |
| `⌘]` | Go forward one note |
| `⌘⌥[` | Jump back one month |
| `⌘⌥]` | Jump forward one month |
| `⌃⌥⌘[` | Jump back one year |
| `⌃⌥⌘]` | Jump forward one year |
| `⌘T` | Jump to current note |
